# AtlasEdu

Sistema de gerenciamento escolar desenvolvido em dupla com foco em aprendizado de desenvolvimento fullstack moderno.

## Objetivo

O projeto AtlasEdu tem como objetivo simular uma plataforma real de gerenciamento escolar, permitindo controle de alunos, professores, turmas, pagamentos e relatórios administrativos através de um dashboard moderno.
